## 🔧 如何使用

### 方案1：直接替换（推荐）
```bash
cd /tmp/parenting-miniprogram
cp data.js data.js.backup  # 备份原文件
cp data_50articles.js data.js  # 替换为新文件
```

### 方案2：合并使用
如果想保留原有文章，需要手动合并：
1. 打开 data.js（原文件，约25篇文章）
2. 打开 data_50articles.js（新文件，50篇文章）
3. 将新文件的articles数组追加到原文件
4. 确保格式正确，语法无误

### 方案3：动态加载
在app.js中动态加载：
```javascript
// 原有文章
const existingArticles = require('./data.js')

// 新增文章
const newArticles = require('./data_50articles.js')

// 合并
const allArticles = [...existingArticles, ...newArticles]
```
