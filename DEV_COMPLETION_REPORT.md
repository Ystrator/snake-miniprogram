### 1. 🌙 夜间模式（最紧急）

**实现内容**：
- ✅ 在首页和文章页添加夜间模式切换按钮（🌙/☀️）
- ✅ 使用CSS变量实现主题切换（`--background-color`, `--text-primary`等）
- ✅ 全局状态管理：`app.globalData.darkMode`
- ✅ 本地存储：`wx.setStorageSync('darkMode')`
- ✅ 支持自动夜间模式（晚上10点-早上7点）
- ✅ 所有页面响应主题变化

**技术实现**：
```javascript
// app.js
setDarkMode(enabled) {
  this.globalData.darkMode = enabled;
  wx.setStorageSync('darkMode', enabled);
  // 通知所有页面更新主题
  const pages = getCurrentPages();
  pages.forEach(page => {
    if (page.onThemeChange) {
      page.onThemeChange(enabled);
    }
  });
}
```

**配色方案**：
- 浅色模式：背景 #F2F2F7，卡片 #FFFFFF，文字 #000000
- 深色模式：背景 #000000，卡片 #1C1C1E，文字 #FFFFFF

**修改文件**：
