# 健康工具功能演示说明

## 📱 功能概览

本次更新新增3个健康工具 + 1个月龄推荐系统，全面提升小程序的实用性和专业性。

---

## 1. 💉 疫苗接种提醒器

### 访问路径
首页 → 健康工具区 → 疫苗提醒

### 核心功能
1. **添加疫苗记录**
   - 疫苗名称（如：乙肝疫苗第一针）
   - 接种日期
   - 备注（可选）

2. **智能状态管理**
   - 自动计算疫苗状态
   - 即将到期（7天内）⚠️
   - 已过期 🔴
   - 已完成 ✅
   - 未来计划 📅

3. **统计面板**
   - 总计疫苗数
   - 已完成数量
   - 即将到期数量（高亮显示）

4. **操作功能**
   - 编辑疫苗信息
   - 删除记录
   - 标记完成/未完成

### 使用场景
- 记录宝宝的疫苗接种计划
- 设置下次接种时间提醒
- 追踪疫苗接种历史

---

## 2. 📊 成长曲线记录

### 访问路径
首页 → 健康工具区 → 成长曲线

### 核心功能
1. **成长数据记录**
   - 身高（cm）
   - 体重（kg）
   - 测量日期

2. **WHO标准对比**
   - 男童/女童标准切换
   - 自动计算月龄
   - 百分比评估
     - <90%: 偏低 🔴
     - 90-95%: 偏瘦 🟡
     - 95-105%: 正常 🟢
     - 105-110%: 偏高 🟡
     - >110%: 超标 🔴

3. **数据展示**
   - 实际数值 vs WHO标准
   - 发育评估标签
   - 测量日期和月龄

### 使用场景
- 定期记录宝宝身高体重
- 对比WHO生长标准
- 监测宝宝发育情况

---

## 3. 🌡️ 发烧护理指南

### 访问路径
首页 → 健康工具区 → 发烧护理

### 核心功能
1. **快速记录体温**
   - 体温数值（℃）
   - 测量日期时间
   - 伴随症状（多选）
     - 咳嗽 🤧
     - 流鼻涕 👃
     - 呕吐 🤮
     - 腹泻 💩
     - 抽搐 ⚠️
     - 呼吸困难 😰
     - 精神萎靡 😴
     - 脱水 💧

2. **智能体温评估**
   - 正常：<37.3℃ 🟢
   - 低热：37.3-38.0℃ 🟡
   - 中热：38.1-39.0℃ 🟠
   - 高热：>39.0℃ 🔴
   - 超高热：>41.0℃ 🚨

3. **自动护理建议**
   - 根据体温等级显示建议
   - 物理降温方法
   - 用药建议
   - 紧急情况提醒

4. **紧急情况识别**
   - 6个月以下高热
   - 体温超过39℃
   - 伴随危险症状（抽搐、呼吸困难等）
   - 自动提醒就医

### 使用场景
- 宝宝发烧时快速记录
- 查看科学的护理建议
- 判断是否需要立即就医
- 追踪体温变化趋势

---

## 4. 📚 内容月龄细分

### 首页月龄推荐
- **智能推荐区**：根据宝宝月龄自动推荐相关文章
  - 0-1岁：新生儿护理、喂养、睡眠
  - 1-3岁：学走路、说话、认知发展
  - 3-6岁：入学准备、社交、情绪管理

### 搜索月龄筛选
- **月龄筛选器**：搜索时可按月龄筛选文章
  - 全部
  - 0-1岁
  - 1-3岁
  - 3-6岁
- **自动选择**：如果有宝宝档案，自动选择对应月龄段

### 使用场景
- 快速找到适合宝宝月龄的内容
- 避免阅读不相关文章
- 提高搜索效率

## 🎯 完整使用流程

### 第一次使用
1. 打开小程序
2. 进入"宝宝档案"页面
3. 设置宝宝信息：
   - 宝宝名字
   - 出生日期
4. 保存后，首页自动显示：
   - 宝宝年龄卡片
   - 月龄专属推荐内容
   - 健康工具入口

### 日常使用

#### 记录疫苗接种
1. 首页 → 疫苗提醒
2. 点击"添加疫苗记录"
3. 填写信息并保存
4. 系统自动提醒即将到期的疫苗

#### 记录成长数据
1. 首页 → 成长曲线
2. 选择男宝/女宝
3. 点击"添加成长记录"
4. 输入身高体重
5. 查看WHO标准对比结果

#### 处理发烧情况
1. 首页 → 发烧护理
2. 点击"记录体温"
3. 输入体温和症状
4. 查看护理建议
5. 根据提示决定是否就医

#### 查找月龄内容
1. 首页查看月龄推荐
2. 或使用搜索功能
3. 选择对应月龄筛选
4. 浏览相关文章

## 💾 数据存储说明

所有数据使用微信小程序本地存储，安全私密：

- **疫苗数据**：`vaccines`
- **成长数据**：`growthRecords` + `babyGender`
- **发烧数据**：`feverRecords`
- **宝宝档案**：`babyInfo`（已存在）

数据持久化保存在用户手机本地，不会上传到服务器，保护隐私。
