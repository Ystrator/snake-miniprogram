# 🎉 育儿知识大全 v1.2.0 发布说明

## 发布日期
2026-02-27

## ✨ 新功能

### 1. 👶 宝宝档案
- 设置宝宝小名和生日
- 自动计算月龄（精确到天）
- 首页显示"宝宝今天X个月X天"
- 根据月龄智能推荐文章

### 2. 🔍 搜索优化
- 搜索结果显示摘要
- 智能相关度排序
- 🔥 最相关标记
- ⭐ 推荐标记
- 相关文章推荐
- 热门搜索词统计

### 3. 📚 文章扩充
- 数据结构优化（新增description字段）
- 文章数量：22篇 → 27篇
- 统一格式标准

## 🔧 技术改进

- **新增页面**: pages/baby/ (宝宝档案)
- **新增API**: 
  - `setBabyInfo()` - 保存宝宝信息
  - `getBabyInfo()` - 获取宝宝信息
  - `calculateBabyAge()` - 计算月龄
  - `getRecommendedCategory()` - 获取推荐分类
- **优化算法**: 搜索相关度智能排序
