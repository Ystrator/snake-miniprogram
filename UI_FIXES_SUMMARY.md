## 修复日期

## 修复的 Bug 列表

### ✅ Bug #1: 搜索功能无法加载文章数据
**文件**: `pages/search/search.js`
**问题**: 搜索页面尝试访问 `knowledgeData.allArticles`，但该数据可能为空或不存在
**修复**: 修改搜索逻辑，从 `knowledgeData.categories` 中提取所有文章
**影响**: 搜索功能现在可以正常工作

### ✅ Bug #2: 文章详情页硬编码数据
**文件**: `pages/article/article.js`
**问题**: 文章数据硬编码在 JS 文件中，与 data.js 的 80 篇文章不同步
**修复**: 重写文章加载逻辑，从 data.js 动态加载文章数据
**影响**: 现在可以查看 data.js 中的所有 80 篇文章

### ✅ Bug #3: 分类页面硬编码数据
**文件**: `pages/category/category.js`
**问题**: 分类页面文章列表硬编码，与 data.js 不同步
**修复**: 从 data.js 动态加载分类数据
**影响**: 分类页面现在显示完整的文章列表

### ✅ Bug #4: 数据格式不兼容
**文件**: `pages/article/article.js`
**问题**: data.js 中的文章使用 `content` 数组，而 WXML 模板期望 `sections` 数组
**修复**: 添加 `convertArticleFormat()` 方法，自动转换数据格式
**影响**: 文章详情页内容可以正确渲染

### ✅ Bug #5: 夜间模式下快速解答板块样式问题
**文件**: `pages/article/article.wxss`
**问题**: 快速解答板块在夜间模式下缺少专用样式
**修复**: 添加夜间模式专用 CSS 样式
**影响**: 夜间模式下可读性显著提升
