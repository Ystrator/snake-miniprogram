## 🔧 如何使用加载组件

### 步骤1：引入组件
在页面 JSON 中添加：
```json
{
  "usingComponents": {
    "loading": "/components/loading/loading"
  }
}
```

### 步骤2：使用组件
在页面 WXML 中添加：
```xml
<loading show="{{isLoading}}" text="加载中..."></loading>
```

### 步骤3：控制显示
在页面 JS 中控制：
```javascript
Page({
  data: { isLoading: false },
  
  loadData() {
    this.setData({ isLoading: true });
    // 数据加载...
    this.setData({ isLoading: false });
  }
});
```
