## 发现的问题

### 🔴 严重问题

#### 1. 分类字段缺失导致筛选功能失效
**问题描述**:
- 数据文件中文章使用 `"category": "0-6个月"` (文本值)
- 代码期望 `"categoryId": "age-0-1"` (ID值)
- 导致分类筛选返回0篇文章

**影响范围**:
- 首页推荐文章功能
- 分类页面文章列表
- 所有依赖分类筛选的功能

**修复方案**:
✅ 为所有80篇文章添加 `categoryId` 字段
✅ 建立分类映射表:
  ```
  "0-6个月"   → "age-0-1" (21篇)
  "6-12个月"  → "age-0-1" (24篇)
  "1-3岁"     → "age-1-3" (19篇)
  "3-6岁"     → "age-3-6" (5篇)
  "通用知识"   → "general" (11篇)
  ```

#### 2. 文章详情页无法加载文章
**问题描述**:
- data.js 中文章ID为 `"article_001"` 格式
- article.js 中硬编码的文章ID为 `"001"` 格式
- ID格式不匹配导致文章查找失败

**影响范围**:
- 所有文章详情页无法加载
- 用户点击文章卡片后看到"文章不存在"提示

**修复方案**:
✅ 重写 article.js，从 data.js 动态加载文章
✅ 支持多种ID格式兼容性查找:
  - 直接匹配: `article_001`
  - 自动转换: `001` → `article_001`
  - 向后兼容旧数据格式
