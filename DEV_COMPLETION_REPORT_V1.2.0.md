## 🔧 技术要点

### 1. 月龄计算算法
```javascript
calculateBabyAge() {
  const birthDate = new Date(this.babyInfo.birthday);
  const today = new Date();
  const days = Math.floor((today - birthDate) / (1000 * 60 * 60 * 24));
  const ageMonths = Math.floor(days / 30);
  const ageDays = days % 30;
  return { months: ageMonths, days: ageDays };
}
```

### 2. 搜索相关度算法
```javascript
const relevance =
  (titleMatch ? 10 : 0) +
  (tagMatch ? 5 : 0) +

  (summaryMatch ? 2 : 0);
```
